AI Project

This project consists of a game in which the map is procedurally generated. The map is divided into biomes, each one with its unique characteristics; different textures for the ground, unique assets for each biome and each one also has a different type of item. The player can pick up these items and if they pick up all the items on the map they win, and can create a new world and repeat the process.

Apart from this, the world is inhabited by different animals, each with its own behaviors, which reproduce and the offspring have different characteristics, causing them to change.

Escape Room

This game consists of escaping from a room with an object. The problem is that as soon as you take the object, the door of the room closes. You will have to find objects that have a similar weight and place them where the other object was so that the door opens again.

As soon as you remove the object from where it is, two doors close. In the room where you are there is an object that you can take and it can open the door to the kitchen (it is not enough weight to open the exit). In the kitchen you will find an object, which added to the one already placed, will be enough to open the exit.
