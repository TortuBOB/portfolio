Escape Room

Technical details

Developed in Unreal

Single-player

Genre: Puzzle

Platform: Computer

Summary

This game consists of escaping from a room with an object. The problem is that as soon as you take the object, the door of the room closes. You will have to find objects that have a similar weight and place them where the other object was so that the door opens again.

As soon as you remove the object from where it is, two doors close. In the room where you are there is an object that you can take and it can open the door to the kitchen (it is not enough weight to open the exit). In the kitchen you will find an object, which added to the one already placed, will be enough to open the exit.

Mechanics

This project was programmed in C++ (without using Unreal's blueprints system).

The player can move around the map and pick up and drop certain objects. The movement is with the "WASD" keys. To pick up an object you have to be looking at it, right click and hold it down. If you release it, you will drop the object.

There is also the pressure plate system. On the table where the lantern is, there is a trigger that detects the weight that is in the table. The two doors are connected to this pressure plate, and only open if there is more than a certain weight on the plate. The door leading to the kitchen opens with less weight than the other one, so when you put the book on the pressure plate only the kitchen door opens. The lantern is the heaviest object, so you need the other two objects to escape.
