2D Platformer

Technical details

Developed in Unity

Single-player

Genre: 2D Platformer

Platform: Computer

Summary

This computer game is a 2D platformer with a system that generates the level as you play, making each time you play different.

The game is infinite in that the level is constantly generated and has no end, the only way to end it is to die.

Mechanics

The player can move around the level with the "A" and "D" keys, for jumping it's the space key. The camera follows the player on the X-axis with an offset so that you can see the platforms better. In addition, the camera has a delay effect when it follows the player's character.

The playable character has a health system, which is shown by a bar in the upper left corner.

In the top right corner the player's score is displayed and below it the number of coins collected. The score is calculated from the coins collected.

The game also has platforms that move horizontally and vertically from one point to another.

Dangers

There are two types of dangers in the game: falling into the void, which kills you instantly, and enemies, which takes some of your life.

Enemies can move in two different ways: up and down, between platforms, to make jumping more difficult, and from right to left, on slightly longer platforms, so that you don't stay on those platforms for too long and force you to move faster to avoid losing life.

Level generation

The level is generated with a system of blocks. The blocks are created in advance (they are prefabs). Each block has a start point, an end point, an exit zone and a death zone.

The start point is the point from which the block is generated. The end point is where the next block is generated. The starting point of a block would be where the end point of the previous block is.

The exit zone is after the end point. Once the player passes through the exit zone, the block they have just passed through will be destroyed and a new one will be created next to the ones already on the map.

The death zone is the fall into the void. It is below the camera's view and kills the player's character if they pass through it.
