NPC Farming

Technical details

Developed in Unreal Engine 4 with blueprints

Platform: Computer

Summary

This project consists of a simulation in which there are several NPCs that have to collect certain resources and take them to the corresponding warehouses. The resources are randomly generated in an area.

Resources

There are three types of resources: green, orange and blue.

There is a resource creation zone, an invisible cube, where resources are generated in random positions within the cube.

Each resource has a different appearance rate, with green appearing more frequently and orange appearing less frequently.

Warehouses

Resources are brought to warehouses by NPCs. Each warehouse accepts different types of resources, with some accepting only one type, others two, and others all three.

They also have a limit on the number of resources they can store and these are distributed evenly depending on the number they accept. If a store is full, it does not accept any more resources.

NPCs

Each NPC can collect only one type of resource, which is shown by the character's colour.

NPCs use the Unreal Engine's EQS system to make decisions.

NPC behaviour consists of finding the nearest resource of the correct type, fetching it, picking it up and taking it to the nearest warehouse that accepts that type of resource.
