3D Physics

Resumen

Este proyecto es el primero en el que se usa la biblioteca de físicas de Bullet. En este proyecto se monta un escenario 3D con objetos que simulan físicas de distintas maneras para aprender a usar esta biblioteca.

Entidades y Sensores

Una entidad es un objeto formado por un cuerpo rígido y un modelo 3D. Hay tres tipos de entidades: cubos, cilindros y esferas.

La parte física de la entidad, el cuerpo rígido (rigidbody), puede ser dinámica, que simula las físicas (gravedad, se mueve al chocar contra algo…), estática, que no se puede mover, y cinemática (kinematic), que no simula las físicas, pero se puede mover.

Un sensor es un objeto que detecta colisiones, pero a diferencia de una entidad, se puede atravesar. Los sensores tienen forma de cubo y pueden tener un modelo o ser invisibles.

Hay una llave que es un sensor, sirve para abrir una puerta. En este caso tiene parte visible (un modelo de una llave) y el cubo que detecta las colisiones es del tamaño justo para ocupar todo el modelo.

Coche

Un coche es un objeto formado por varias entidades. El cuerpo es un cubo y luego tiene cuatro ruedas que son cilindros. También tiene dos cilindros entre el cuerpo y las ruedas delanteras, mientras que las ruedas de atrás están unidas al coche directamente. Todas estas entidades son dinámicas.

El coche se puede controlar con “WASD”. Cuando el usuario presiona la tecla “W” todas las ruedas comenzaran a rotar sobre sí mismas, y gracias al rozamiento con el suelo se mueve el coche. Con la “S” rotan, pero en el sentido contrario para desplazarse hacia atrás. Las teclas “A” y “D” hacen que giren los cilindros delanteros que unen el cuerpo con las ruedas, haciendo que las ruedas estén giradas, y así el coche avanza hacia la derecha o la izquierda.

Puerta y Plataforma

La puerta está formada por una entidad en forma de cubo dinámica, y un cilindro estático. Ambos están unidos y cuando el usuario coge la llave se activa un motor que hace que la puerta gire sobre el cilindro abriéndose.

La plataforma, que sirve para cruzar un agujero, está formada por un cubo cinemático y un sensor invisible. El sensor está encima de la plataforma y cuando el coche lo toca, la plataforma cruza el agujero.

Escena

La escena se encarga de crear la cámara y la luz. Aparte de esto genera todo el escenario, el suelo, las paredes, y el resto de objetos.

La escena también se encarga de comprobar las colisiones entre los sensores y los objetos correspondientes, y realiza las acciones pertinentes cuando lo detecta.
